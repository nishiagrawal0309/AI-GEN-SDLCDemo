# Requirement: AI SiteMap Redirect Resolution

## Description
Replace 301/302 links from the sitemaps XML (sample_sitemap.xml) with Final destination 
and remove 404 links.At the end generate a CSV file with all the
301/302 URLs and their final URL status at given path.

## File Details
- File Name: sample_sitemap.xml
- Path: /web/pages/

## Acceptance Criteria
- Proper CSV file should be generated at provided path
- CSV should have header columns ("original_url","final_url","status_code", "is_redirect","redirect_type","redirect_chain","error","checked_at")
